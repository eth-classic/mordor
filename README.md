### Parity-Ethereum

Minimum required version: `v2.7.1`

```
parity --chain mordor
```

### Multi-Geth

Minimum required version: `v1.9.7`

```
geth --mordor
```

### Hyperledger Besu

Minimum required version: `v1.4.0`

```
besu --network mordor
```

### ~~Geth Classic~~

Minimum required version: `v6.1.2` (deprecated)

```
geth --chain mordor
```

Geth Classic reached end of life and is no longer recommended.
